À propos

Le site « Vue sur les contrats » est un outil de visualisation qui permet de consulter les contrats et les subventions octroyés par la Ville de Montréal de façon simple et conviviale. Il donne accès à tous les contrats autorisés par le comité exécutif et par le conseil municipal depuis janvier 2012.

Vue sur les contrats utilise les fichiers disponibles sur le portail des données ouvertes de la Ville de Montréal (http://donnees.ville.montreal.qc.ca/). Ces fichiers proviennent de diverses sources, notamment :

  • l’outil de gestion des dossiers décisionnels utilisé par les élus
  • les systèmes financiers de la Ville.

Ces systèmes sont utilisés pour des besoins internes et certaines informations souhaitables du point de vue de la transparence, comme le numéro d’entreprise, ne sont pas disponibles pour le moment. De plus, la saisie des informations peut varier d’un contrat à l’autre.

Les citoyens qui veulent pousser leurs recherches plus loin ont accès à une interface de programmation (« API ») qui permet d’automatiser les requêtes (http://ville.montreal.qc.ca/vuesurlescontrats/api/). La documentation de l'API est également disponible dans le portail des données ouvertes (http://donnees.ville.montreal.qc.ca/dataset/contrats-et-subventions-api/).

Ce projet s’inscrit dans le plan d’action de Montréal, ville intelligente et numérique (http://villeintelligente.montreal.ca/).
